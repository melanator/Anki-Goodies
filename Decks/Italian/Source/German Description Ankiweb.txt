In sieben Monaten lernt er, jederzeit italienische Verben zu kombinieren, von der Gegenwart bis zur unvollkommenen Unterjunctive Vergangenheit.

→ Erwerben Sie die richtige Konjugation schnell, ist eine Supermacht
→ Sie können dieses Deck sogar verwenden, bevor Sie das italienische Studium formal beginnen
→ Das Deck enthält 60 Verben, einschließlich aller unregelmäßigen Verben, die zwischen den 50 häufigsten italienischen Verben aufgeführt sind
→ Unregelmäßige Verben folgen bestimmten Mustern: Verben im Deck umfassen alle unregelmäßigen Konjugationsmuster, die auf Italienisch gefunden werden
